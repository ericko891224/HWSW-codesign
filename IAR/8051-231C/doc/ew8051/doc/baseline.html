BaseLine Edition

The IAR BaseLine Edition is a special low-priced version of the IAR Embedded Workbench. It has limitations both in performance and in the service and support that is provided.

The limitations of the BaseLine Edition are as follows:

  • The IAR BaseLine C compiler supports a maximum of 8 KB generated code, and has no support for floating-point arithmetic. It will not generate any assembly code output.
  • The IAR Assembler delivered is the full version without any restrictions.
  • The IAR XLINK Linker will link a maximum of 8 KB originating from C souce code, and an unlimited amount of code originating from assembly code.
  • The IAR BaseLine C-SPY Simulator will read a maximum of 8 KB code.

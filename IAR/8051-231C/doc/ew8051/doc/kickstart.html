Kickstart Edition

The IAR Kickstart Edition of the IAR Embedded Workbench is a special version designed for starter kits. It has limitations both in performance and in the service and support that is provided. In this version, the ROM-monitor version of the C-SPY debugger is also included.

The limitations of the Kickstart Edition are as follows:

  • The IAR Kickstart C compiler supports a maximum of 2 KB generated code. It will not generate any assembly code output.
  • The IAR Assembler delivered is the full version without any restrictions.
  • The IAR XLINK Linker will link a maximum of 2 KB originating from C source code, and an unlimited amount of code originating from assembly code.
  • The IAR Kickstart C-SPY drivers will read a maximum of 2 KB code.

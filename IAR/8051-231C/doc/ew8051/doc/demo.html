Demo version

The limitations of the demo version are as follows:

  • Output from the tools can only be read by the demo version of the IAR C-SPY Simulator Debugger for 8051.

  • The IAR Demo C compiler supports a maximum of 2 KB generated code. It will not generate any assembly code output.

  • The IAR Demo Assembler supports a maximum of 1 KB generated code.

  • The IAR Demo XLINK Linker will link a maximum of 2 KB source code.

  • The IAR Demo C-SPY drivers will read a maximum of 2 KB code.

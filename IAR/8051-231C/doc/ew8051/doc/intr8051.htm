Important information

  • About the intrinsic library

    The intrinsic library is entirely written in 8051 assembly language, except for functions that emulate C functions. Currently, only C-SPY versions of the "putchar" and "getchar" functions are written in assembly, as well as the "setjmp" and "longjmp" functions. The library modules are called L00, L01, etc.

  • Contents of the intrinsic library

    The ICC8051 intrinsic library consists of these assembler source files:

    • L00.s03
      Binary/unary char & int operations.
    • L00_2.s03
      Like L00.s03, but not for the ACALL library.
    • L01.s03
      Binary/unary long operations.
    • L01_2.s03
      Like L01.s03, but not for the ACALL library.
    • L02.s03
      Binary/unary float operations.
    • L03.s03
      Reserved.
    • L04.s03
      Bit field support.
    • L05.s03
      switch" operations.
    • L06.s03
      setjmp" & "longjmp".
    • L07.s03
      Reserved.
    • L08.s03
      Inline-routines like strcmp, etc.
    • L16.s03
      Binary/unary char & int operations.
    • L17.s03
      Adressing support.
    • L17_2.s03
      Like L17.s03, but not for the ACALL library.
    • L18.s03
      Bank-switching support.
    • L20.s03
      Shortcuts only.
    • L20_2.s03
      Like L20.s03, but not for the ACALL library.
    • Debug.s03
      C-SPY support.
    • Debug_LJ.s03
      C-SPY support for longjump.
    • Recurse.s03
      Recursion handler (XLINK invoked).
